C++ is a high-level, general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes". It has imperative, object-oriented and generic programming features. C++ runs on lots of platform like Windows, Linux, Unix, Mac etc.

Learn Java

Java is a write-once, run-anywhere programming language developed by Sun Microsystems. You can combine Java with a lot of technologies like Spring, node js, Android, Hadoop, J2EE, etc… Java was originally developed by James Gosling with his colleagues at Sun Microsystems during the early 1990s.
